MySQL jest relacyjnym systemem zarządzania bazą danych.

Projekt został opracowany przez firmę MySQL AB.

Obecnie rozwijany jest przez Oracle Corporation.

MySQL należy do najpopularniejszych systemów baz danych na świecie.

Jest wykorzystywany zarówno w niewielkich aplikacjach, jak i w dużych systemach produkcyjnych.

MySQL jest dostępny w wersji Community Edition rozwijanej jako oprogramowanie otwartoźródłowe.

Dostępne są również komercyjne edycje Enterprise oferujące dodatkowe funkcje administracyjne oraz bezpieczeństwa.

MySQL obsługuje relacyjny model danych.

Dane przechowywane są w tabelach.

Tabele składają się z wierszy oraz kolumn.

Możliwe jest definiowanie kluczy głównych.

Możliwe jest definiowanie kluczy obcych.

Możliwe jest tworzenie relacji pomiędzy tabelami.

MySQL obsługuje standardowy język SQL.

Programiści mogą wykonywać operacje INSERT.

Programiści mogą wykonywać operacje UPDATE.

Programiści mogą wykonywać operacje DELETE.

Programiści mogą korzystać z operacji JOIN.

Programiści mogą tworzyć widoki.

Programiści mogą tworzyć indeksy.

Programiści mogą tworzyć procedury składowane.

Programiści mogą tworzyć funkcje.

Programiści mogą tworzyć wyzwalacze.

Znajomość SQL pozwala bardzo szybko rozpocząć pracę z MySQL.

Najczęściej wykorzystywanym silnikiem magazynowania danych jest InnoDB.

InnoDB obsługuje transakcje.

InnoDB wspiera klucze obce.

InnoDB zapewnia zgodność z właściwościami ACID.

Atomicity oznacza niepodzielność transakcji.

Consistency oznacza zachowanie spójności danych.

Isolation oznacza izolację równocześnie wykonywanych transakcji.

Durability oznacza trwałość zapisanych danych.

MySQL oferuje również inne silniki magazynowania danych.

Jednym z nich jest MyISAM.

MyISAM zapewnia bardzo szybkie operacje odczytu.

MyISAM nie obsługuje jednak transakcji ani kluczy obcych.

Optymalizator analizuje strukturę zapytania.
